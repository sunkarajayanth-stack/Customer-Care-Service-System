# Customer Care Service System

This project is a simple web-based customer service portal that allows users to submit support tickets.

## Features
- Submit customer issues using a form
- View submitted tickets dynamically
- Remove tickets by clicking them
- Fetch customer support tips using API
- Responsive layout for mobile and desktop

## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API

## Course Outcomes Covered

CO1 – HTML structure, Internet concepts, CSS  
CO2 – Forms, Semantic HTML, Responsive layout  
CO3 – JavaScript logic, functions, arrays, objects  
CO4 – DOM manipulation, events, Fetch API  
CO5 – GitHub repository and deployment
